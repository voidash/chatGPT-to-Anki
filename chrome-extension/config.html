Prompt Customization

Customize the base prompt used to generate flashcards. This works together with the modal form options - the modal settings will be added as "Additional Instructions" to this prompt.

Modal Integration:
• Your custom prompt is used as the base
• The selected conversation content is automatically added
• Modal form settings (topic focus, difficulty, etc.) are added as "Additional Instructions"
• All components work together to create the final flashcard generation prompt

Context Prompt Settings

Customize the prompt used when generating flashcards from collected context items. Use {CONTEXT} placeholder where you want the collected text to be inserted.

Context Collection Process:
• Right-click selected text and choose "Add to Context for Flashcards"
• Text is saved with source information (page title, URL, timestamp)
• Use the popup "Generate Flashcards" button to create flashcards
• The system opens ChatGPT with your context prompt and collected information
• All collected context items are combined and inserted at {CONTEXT} placeholder
